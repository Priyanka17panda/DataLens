# 📊 DataLens – Dataset Analyser Assistant

## Project Overview

DataLens is an AI-powered web application designed to simplify data exploration and analysis. It enables users to upload CSV or Excel datasets and instantly receive comprehensive insights without writing a single line of code. The application automates Exploratory Data Analysis (EDA), generates interactive visualizations, identifies data quality issues, and provides intelligent recommendations to help users better understand their data.

Beyond traditional analytics, DataLens integrates a conversational AI chatbot that allows users to interact with their dataset using natural language. Users can ask questions about trends, statistics, missing values, distributions, and other dataset characteristics, making data analysis more accessible to students, researchers, analysts, and business professionals.

### Key Features

* Upload and analyze CSV and Excel datasets.
* Automated Exploratory Data Analysis (EDA).
* Interactive charts and visualizations for better data understanding.
* Missing value, duplicate, and data quality detection.
* Statistical summaries and feature-wise insights.
* Intelligent preprocessing recommendations.
* Machine learning model suggestions based on dataset characteristics.
* AI-powered chatbot for natural language dataset queries.
* Clean, responsive, and user-friendly interface.

### Technologies Used

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** FastAPI (Python)
* **Libraries:** Pandas, NumPy, Plotly, Matplotlib, Seaborn, OpenPyXL
* **AI Integration:** Google Gemini API
* **Deployment:** Render (Backend) and Vercel/Netlify (Frontend)

### Motivation

Data analysis often requires technical expertise and familiarity with multiple tools. DataLens was developed to bridge this gap by providing an intuitive platform where users can upload a dataset and receive meaningful insights within seconds. By combining automated analytics with conversational AI, the project makes data exploration faster, more interactive, and accessible to users with varying levels of technical knowledge.

### Future Scope

Future enhancements include user authentication, PDF report generation, SQL database connectivity, model training within the platform, time-series analytics, cloud storage integration, and support for comparing multiple datasets.
